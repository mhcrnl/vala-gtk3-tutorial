SearchEntry
===========
The SearchEntry widget is graphically similar to the :doc:`entry`, however it is tailored for search uses such as finding the name in an address book or a particular word in a text document. It displays additional icons within the SearchEntry for initiating the search and clearing the criteria.

===========
Constructor
===========
Construction of the SearchEntry widget is made using::

  var searchentry = new Gtk.SearchEntry();

=======
Methods
=======
.. note::

  The SearchEntry inherits many methods from the Entry widget. Please see the methods section of the Entry widget for methods to use.
